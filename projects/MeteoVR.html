Project Description

This project was the central theme of our course. The goal was to learn how to use Unity VR and implement it in a project context. In this case, it was a VR weather app. My classmate and I decided to create an app that lets users select a city’s current weather from a predefined list and change the weather in the virtual space accordingly. We did this using a free weather API and a VFX plugin to simulate the weather within the app.

My Role & Contributions

As a programmer with the necessary equipment at home, I've integrated VR and interactive elements in the scene. This includes the UI, which uses the weather script to change the weather.

  • Implement VR features
  • Add VR interaction elements
  • Implement a UI linked to the weather script

Challenges & Learnings

My main challenge remains VR itself and its integration. Even though our instructor provided us with all the necessary documentation to implement certain concepts, such as Unity’s fixed UI, integrating VR can sometimes be a bit tricky. But overall I’ve learned a lot about Unity and OpenXR, and I’m confident that I can integrate VR into a project without any issues. Another issue my colleague and I encountered was optimizing the app for VR headsets, specifically to ensure it ran smoothly on the Meta Quest 2 and Meta Quest 3S. This required us to modify our map asset, which contained too many polygons for the headset to handle.
